# 논리연산자
> 자바스크립트엔 세 종류의 논리 연산자 ||(OR), &&(AND), !(NOT)이 있다.
<br/>

> 연산자에 '논리’라는 수식어가 붙긴 하지만 논리 연산자는 피연산자로 불린형뿐만 아니라 모든 타입의 값을 받을 수 있다
> 연산 결과 역시 모든 타입이 될 수 있다.
<br/>

## || (OR)
- ‘OR’ 연산자는 두 개의 수직선 기호로 만들 수 있다.
    ~~~
    result = a || b;
    ~~~
- 전통적인 프로그래밍에서 OR 연산자는 불린값을 조작하는 데 주로 쓰인다. 인수 중 하나라도 true이면 true를 반환하고, 그렇지 않으면 false를 반환
- 자바스크립트의 OR 연산자는 다루긴 까다롭지만 강력한 기능을 제공한다.
- OR 연산자는 이항 연산자이므로 아래와 같이 네 가지 조합이 가능.
    ~~~
    alert( true || true );   // true
    alert( false || true );  // true
    alert( true || false );  // true
    alert( false || false ); // false
    ~~~
- 피연산자가 모두 false인 경우를 제외하고 연산 결과는 항상 true.
- 피연산자가 불린형이 아니면, 평가를 위해 불린형으로 변환.
- 예를 들어, 연산 과정에서 숫자 1은 true로, 숫자 0은 false로 바뀜
    ~~~
    if (1 || 0) { // if( true || false ) 와 동일하게 동작합니다.
        alert( 'truthy!' );
    }
    ~~~
- OR 연산자 ||은 if문에서 자주 사용됨.
- 주어진 조건 중 하나라도 참인지를 테스트하는 용도로 많이 사용.

## 첫 번째 truthy를 찾는 OR 연산자 ‘||’
- 지금까진 피연산자가 불린형인 경우만을 다뤘는데, 이제 자바스크립트에서만 제공하는 논리연산자 OR의 '추가’기능을 설명한다.
- OR 연산자와 피연산자가 여러 개인 경우:
    ~~~
    result = value1 || value2 || value3;
    ~~~
- OR ||연산자는 다음 순서에 따라 연산을 수행.
    1. 가장 왼쪽 피연산자부터 시작해 오른쪽으로 나아가며 피연산자를 평가.
    1. 각 피연산자를 불린형으로 변환. 변환 후 그 값이 true이면 연산을 멈추고 해당 피연산자의 변환 전 원래 값을 반환.
    1. 피연산자 모두를 평가한 경우(모든 피연산자가 false로 평가되는 경우)엔 마지막 피연산자를 반환.
- ※ 여기서 핵심은 반환 값이 형 변환을 하지 않은 원래 값이라는 것.

## && (AND)
- 두 개의 앰퍼샌드를 연달아 쓰면 AND 연산자 &&를 만들 수 있다.
    ~~~
    result = a && b;
    ~~~
- 전통적인 프로그래밍에서 AND 연산자는 두 피연산자가 모두가 참일 때 true를 반환. 그 외의 경우는 false를 반환.

## ! (NOT)
- 논리 연산자 NOT은 느낌표 !를 써서 만들 수 있다.
- NOT 연산자의 문법은 매우 간단.
    ~~~
    result = !value;
    ~~~
- NOT 연산자는 인수를 하나만 받고, 다음 순서대로 연산을 수행.
    1. 피연산자를 불린형(true / false)으로 변환.
    1. 1에서 변환된 값의 역을 반환.
- NOT을 두 개 연달아 사용(!!)하면 값을 불린형으로 변환할 수 있다.
